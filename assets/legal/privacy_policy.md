## 1. Information We Collect

### 1.1 Account Information
When you create an account, we collect:

### 1.2 Note Content
We store the notes you create within the application, including:
- Note titles and text content (in Markdown format)
- Images and media attached to notes
- Note metadata (creation date, modification date, visibility settings, color labels)
- Tags and categories

### 1.3 Social Information
If you use social features, we collect:
- Friend connections and group memberships
- Shared note recipients
- Favorite and interaction data

### 1.4 Usage Data
We automatically collect certain information when you use the app:
- Device type, operating system, and version
- App version
- Language preferences
- General usage patterns and feature interactions
- Crash reports and performance data

### 1.5 Information We Do NOT Collect
- We do **not** collect location data
- We do **not** read or scan your private notes for advertising
- We do **not** collect contacts from your device

## 3. Data Storage and Security

### 3.1 Storage
Your data is stored securely using **Supabase** infrastructure, which is built on:
- **PostgreSQL** databases hosted in the **Frankfurt (EU)** region
- Industry-standard encryption at rest and in transit (TLS 1.2+)
- Row Level Security (RLS) policies to ensure data isolation

### 3.2 Media Storage
Images and media files are stored in **Supabase Storage** with:
- Automatic optimization and compression
- Secure, signed URLs for access
- Bucket-level access control policies

### 3.3 Local Storage
For offline access, some data is cached locally on your device using an encrypted local database. This data remains on your device and is cleared when you log out.

### 3.4 Security Measures
We implement appropriate technical and organizational measures to protect your personal data, including:
- Encrypted data transmission (HTTPS/TLS)
- Encrypted data storage
- Row Level Security (RLS) database policies
- Regular security audits and updates
- Secure authentication via Supabase Auth
